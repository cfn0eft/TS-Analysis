# TS Analysis

**[ZIPをダウンロード](https://github.com/cfn0eft/TS-Analysis/releases/download/v1.1/TS.Analysis-1.1.zip)**
**[開発コード](https://github.com/cfn0eft/TS-Analysis_dev)**

## 1. このアプリについて
TS Analysis は、複数の時系列解析手法を同一条件で比較 検証するための Windows 用 GUI アプリです。
教育 研究用途を目的としており、投資判断を行うものではありません。
Python や専門的な知識は不要です。

## 2. 起動方法 重要
1. フォルダ TS Analysis を開きます
2. TS Analysis.exe をダブルクリックします

Python や uv のインストールは不要です。

## 3. 基本的な使い方
1. Data ページで CSV ファイルを読み込むか Yahoo Finance から取得します
2. 左側メニューから分析手法を選択します
3. パラメータを設定して Run をクリックします
4. 結果を確認し PNG や CSV を保存します

## 4. 利用できる分析手法

### 移動平均クロス MAクロス
- 短期と長期の移動平均線による売買シグナルを表示します
- Buy and Hold との比較ができます
- 戦略パフォーマンス指標を確認できます

### フーリエ解析
- データに含まれる周期構造を可視化します
- データは等間隔であることを前提とします
- 周期性の把握を目的とします

### ウェーブレット解析
- 時間と周波数を同時に解析します
- 周期が時間とともに変化する様子を確認できます
- スケール設定により見え方が変化します

### 多項式回帰
- トレンドの可視化を目的とした手法です
- 将来予測の正確性を保証するものではありません
- 過学習の例としての利用を想定しています

### ARIMA モデル
- 線形性と定常性を前提とした時系列モデルです
- RMSE や MAE などの評価指標を表示します
- パラメータ設定により結果が大きく変化します

## 5. 実行結果の保存について
実行ごとに run_id が自動的に発行されます。
結果は run_id ごとに整理されて保存されます。

保存先例
runs\run_id\

保存される内容
- PNG グラフ画像
- CSV 指標データ UTF 8 SIG
- run_context.json 作成される場合があります

## 6. 実行ログ
実行履歴は run_history.jsonl に保存されます。
後から条件と結果を確認 再現できます。

## 7. 保存場所のルール
通常は TS Analysis.exe と同じフォルダに保存されます。
書き込みできない場合は LOCALAPPDATA\nitiana_app に自動退避します。
実際の保存先は run_history.jsonl の artifacts_dir を確認してください。

## 8. ネットワークについて
Yahoo Finance からの取得にはネット接続が必要です。
接続できない場合は画面下に警告が表示されます。
アプリは終了しません。

## 9. 表示について
日本語表示のため NotoSansJP フォントを同梱しています。
文字化けは発生しません。

## 10. トラブルシューティング
起動しない場合
- ウイルス対策ソフトのブロックを確認してください
- フォルダを信頼済みの場所に移動してください

保存できない場合
- フォルダの書き込み権限を確認してください
- artifacts_dir を確認してください
